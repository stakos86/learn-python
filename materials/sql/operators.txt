Основные операторы DDL (Data Definition Language):
CREATE: Создание новых объектов базы данных (таблиц, индексов и т.д.)
ALTER: Изменение существующих объектов базы данных
DROP: Удаление объектов базы данных
Основные операторы DML (Data Manipulation Language):
SELECT: Выбор данных из таблицы
INSERT: Вставка новых данных в таблицу
UPDATE: Обновление существующих данных в таблице
DELETE: Удаление данных из таблицы
Операторы TCL (Transaction Control Language):
BEGIN TRANSACTION: Начало транзакции
COMMIT: Подтверждение транзакции
ROLLBACK: Откат транзакции
SAVEPOINT: Фиксация промежуточного состояния транзакции
Логические операторы:
AND: Логическое "И"
OR: Логическое "ИЛИ"
NOT: Отрицание условия
IN: Проверка наличия значения в списке
LIKE: Поиск подстроки в тексте
BETWEEN: Диапазон значений
EXISTS: Проверка наличия записей в подзапросе
Сравнительные операторы:
=: Равно
<>, !=: Не равно
>: Большее
<: Меньше
>=, <=: Большее/меньше или равно
IS NULL: Проверка на NULL
Арифметические операторы:
+: Сложение
-: Вычитание
*: Умножение
/: Деление
%: Остаток от деления
Другие полезные операторы:
GROUP BY: Группировка результатов запроса
HAVING: Фильтрация группированных результатов
ORDER BY: Сортировка результатов запроса
LIMIT: Ограничение количества возвращаемых строк
