Открыть БД : .open адрес файла
Закрыть БД : .exit или control+С


Базовые команды
.help - показать список всех доступных команд
.exit или .quit - выйти из интерактивного режима
.databases - список подключенных баз данных
.tables - список таблиц в текущей базе данных
.schema TABLE_NAME - показать структуру таблицы
Настройка вывода
.mode MODE - установить режим вывода (list, column, html и т.д.)
.headers ON|OFF - включить/выключить заголовки при выводе результатов запросов
.width NUM NUM ... - задать ширину столбцов для режима column
.separator STRING - установить разделитель для вывода в режиме list
Импорт/экспорт данных
.import FILE TABLE_NAME - импортировать данные из файла в таблицу
.dump - экспортировать структуру и данные базы в SQL-скрипт
Другие полезные команды
.read FILE - выполнить SQL из файла
.timeout MS - установить таймаут для подключения к базе данных
.stats ON|OFF - включить/выключить статистику запросов
.timer ON|OFF - включить/выключить измерение времени выполнения запросов
Команды для работы с данными
SELECT ... - выборка данных
INSERT INTO ... - вставка данных
UPDATE ... - обновление данных
DELETE FROM ... - удаление данных
